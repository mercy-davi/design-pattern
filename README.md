# design-pattern
design-pattern 内容定位
1、用设计模式能帮助我们更好地解决实际问题。
2、学习前人是如何总结经验的，把经验为自己所用。
3、培养将业务需求转换为技术实现的能力。

为什么要学习设计模式？
1、写出优雅的代码。
2、更好地重构项目。
3、经典框架都在用设计模式解决问题。

Spring中用到的设计模式:
1、工厂模式  BeanFactory
2、装饰器模式  BeanWrapper
3、代理模式  AopProxy
4、单例模式  ApplicationContext
5、委派模式  DispatcherServlet
6、策略模式  HandlerMapping
7、适配器模式  HandlerAdapter
8、模板方法模式  JdbcTemplate
9、观察者模式  ContextLoaderListener
...
